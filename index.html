Dan's Cheat Sheet

Tags To Remember

Tag

Name

Description
<table> Table The wrapper element for all HTML tables.
<thead> Table Head The set of rows defining the column headers in a table.
<tbody> Table Body The set of rows containing actual table data.
<tr> Table Row The table row container.
<td> Table Data The table row container.
<tfoot> Table Foot The set of rows defining the footer in a table.

Atributes To Remember

Attribute

Name

Description
padding Padding An element's padding area is the space between its content and its border.
margin Margin This property can be used to set a margin on all four sides of an element.
text-align Text Alignement The text-align CSS property sets the horizontal alignment of the content inside a block element or table-cell box.
background-color Background Color The background-color CSS property sets the background color of an element.
color Color The color CSS property sets the foreground color value of an element's text and text decorations, and sets the value.
font-weight Font Weight The font-weight CSS property sets the weight (or boldness) of the font.
